# Ferrari 250 GTO - Real World Asset (RWA) Tokenization

This repository contains the proof-of-concept (PoC) smart contract for the tokenization of a physical 1962 Ferrari 250 GTO.

## Technical Specifications
- **Standard:** ERC-20 Fungible Token
- **Extensions:** ERC20Burnable (To support the physical asset buyout-and-burn mechanism)
- **Total Supply:** 50,000,000 `$GTO`
- **Network:** Ethereum Sepolia Testnet

## Deployment Proof
The smart contract has been successfully deployed and verified on the Sepolia Testnet.
